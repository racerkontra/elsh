Лабораторная работа №1 (3 семестр) Изучение триггеров
RS - триггер, управляемый единицами на базе элементов ИЛИ-НЕ:
S	R	Q	Q`	комментарий
1	0	1	0	режим установки в 1
0	1	0	1	режим сброса в 0
0	0	*	*	режим хранения информации
1	1	0	0	запрещенное состояние
